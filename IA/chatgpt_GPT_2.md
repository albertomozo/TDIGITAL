# 🧩 1. 3 GPTs completos listos para usar en clase

## 🤖 GPT 1: Analista de Procesos Empresariales

### 🎯 Objetivo
Analizar procesos y proponer mejoras en clave de Transformación Digital.

---

### ⚙️ Configuración

**Nombre:** Analista de Procesos  
**Rol:** Consultor en transformación digital  

---

### 🧠 Instrucciones

```

Eres un consultor experto en transformación digital.

Tu función es analizar procesos empresariales y proponer mejoras.

Siempre debes responder con esta estructura:

1. Descripción del proceso
2. Problemas detectados
3. Propuestas de mejora
4. Nivel de impacto (bajo, medio, alto)
5. Posibles herramientas digitales

Usa lenguaje claro y orientado a negocio.

```

---

### 📌 Ejemplo de uso

```

Analiza este proceso:
Una empresa recibe pedidos por email y los gestiona manualmente en Excel.

```

---

### 💡 Resultado esperado
- Identificación de ineficiencias  
- Propuestas de automatización  
- Visión estratégica

## 📧 GPT 2: Gestor de Atención al Cliente

### 🎯 Objetivo
Clasificar consultas y generar respuestas estandarizadas.

---

### ⚙️ Configuración

**Nombre:** Asistente de Atención al Cliente  
**Rol:** Agente de soporte  

---

### 🧠 Instrucciones

```

Eres un asistente de atención al cliente.

Debes analizar cada mensaje recibido y responder con:

1. Tipo de consulta (información, incidencia, queja, etc.)
2. Nivel de prioridad (bajo, medio, alto)
3. Respuesta sugerida (clara y profesional)
4. Acción recomendada

Mantén un tono profesional y empático.

```

---

### 📌 Ejemplo de uso

```

Mensaje:
"No he recibido mi pedido y necesito una solución urgente"

```

---

### 💡 Resultado esperado
- Clasificación automática  
- Respuestas consistentes  
- Mejora en tiempos de respuesta

# 📊 2. Comparativa: GPT vs Automatización con herramientas externas

## 🧠 Enfoque conceptual

| Aspecto | GPTs | Automatización (Make, Zapier, etc.) |
|--------|------|------------------------------------|
| Tipo | Inteligencia | Ejecución |
| Función | Pensar, analizar, generar | Ejecutar acciones |
| Complejidad | Baja-media | Media-alta |
| Necesidad técnica | Baja | Media |
| Tiempo de implementación | Rápido | Medio |
| Escalabilidad | Conceptual | Real (automatizada) |

---

## 🔁 Ejemplo práctico

### Caso: gestión de emails

#### 🧠 Con GPT
- Analiza email  
- Genera respuesta  
- Propone acción  

👉 Requiere intervención humana  

---

#### ⚙️ Con automatización
- Recibe email  
- Clasifica automáticamente  
- Envía respuesta  
- Guarda en base de datos  

👉 Proceso automático completo  

---

## 💡 Conclusión

- GPT → cerebro  
- Automatización → manos  

👉 Juntos forman un sistema completo de Transformación Digital

# 🧪 3. Proyecto práctico: Diseñar un sistema con IA

## 🎯 Objetivo del proyecto

Diseñar un sistema que optimice un proceso empresarial usando:
- GPT (inteligencia)
- Opcional: automatización (conceptual)

---

## 🏢 Paso 1: Definir el problema

Ejemplos:
- Gestión de emails  
- Atención al cliente  
- Organización de tareas  
- Generación de informes  

---

## 🔍 Paso 2: Analizar el proceso actual

Los alumnos deben describir:

- Cómo se hace actualmente  
- Qué problemas tiene  
- Cuánto tiempo consume  

---

## 🤖 Paso 3: Diseñar un GPT

Deben definir:

- Nombre  
- Rol  
- Instrucciones  
- Tipo de respuestas  

---

## 🔁 Paso 4: Definir el nuevo flujo

Ejemplo:

1. Entrada (email, formulario, etc.)  
2. GPT analiza  
3. GPT genera respuesta  
4. Usuario valida o ejecuta  

---

## ⚙️ Paso 5: Propuesta de automatización (opcional)

Ejemplo:

- Formulario → IA → Base de datos  
- Email → IA → Respuesta automática  

---

## 📊 Paso 6: Evaluación del impacto

Deben analizar:

- Tiempo ahorrado  
- Mejora de calidad  
- Escalabilidad  
- Reducción de errores  

---

## 📋 Entregable final

Documento con:

1. Descripción del problema  
2. Proceso actual  
3. GPT diseñado  
4. Nuevo flujo de trabajo  
5. Beneficios  

---

## 🧠 Rúbrica de evaluación (opcional)

| Criterio | Puntuación |
|---------|----------|
| Definición del problema | /10 |
| Diseño del GPT | /10 |
| Claridad del proceso | /10 |
| Innovación | /10 |
| Aplicación real | /10 |

---

## 🏁 Mensaje final para alumnos

> La Transformación Digital no consiste en usar herramientas  
> sino en rediseñar cómo funciona el trabajo  

👉 GPTs = primer paso  
👉 Automatización = siguiente nivel  
👉 Sistema completo = objetivo final
