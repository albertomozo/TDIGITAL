# Datos en Informática

## ¿Qué es un dato?
Un dato es una unidad mínima de información que puede ser almacenada, procesada y transmitida por sistemas digitales. En informática, los datos son valores que representan hechos, mediciones, eventos o acciones, como un número, una fecha, el nombre de un cliente o el resultado de una encuesta.

## ¿Por qué importan los datos?
- Permiten describir la realidad de un proceso o una operación.
- Son la base para tomar decisiones más precisas y rápidas.
- Transforman observaciones aisladas en información útil cuando se juntan y se analizan.

## Datos en informática: algunos ejemplos
- Fecha y hora de una venta.
- Producto vendido y cantidad.
- Resultado de una encuesta de satisfacción.
- Precio de un insumo.

Estos datos pueden almacenarse en tablas, archivos CSV, hojas de cálculo, bases de datos o sistemas en la nube.

## Evolución del concepto de datos
1. **Datos aislados**: antes se registraban en papel o en archivos escritos de forma manual.
2. **Datos digitales simples**: llegaron las hojas de cálculo y las bases de datos básicas.
3. **Datos integrados**: se empezó a cruzar información entre ventas, compras y satisfacción para entender resultados.
4. **Datos analíticos**: se usaron dashboards y gráficos para visualizar patrones y tendencias.
5. **Datos inteligentes**: hoy la IA, el análisis avanzado y las plataformas conectadas permiten extraer conclusiones automáticas.

![imagen datos evolucion](datos-evolucion.png)

## El futuro de los datos
- **Cultura data-driven**: las organizaciones toman decisiones basadas en datos en lugar de intuiciones.
- **Automatización**: los datos se recolectan y se integran automáticamente desde múltiples fuentes.
- **IA y analítica avanzada**: los modelos aprenden de los datos y generan recomendaciones, predicciones y alertas.
- **Ética y privacidad**: el manejo responsable de los datos es esencial para proteger a las personas y cumplir normas.

![imagen datos futuro](datos-futuro.png)

## Ejemplo práctico en este repositorio
Un caso concreto que muestra este concepto está en:

- `examples/dashboard-cafe/inicio.md`

Allí se explica cómo una cafetería recolecta datos de tres fuentes:
- encuestas de satisfacción,
- ventas diarias en Excel/CSV,
- compras de proveedores.

Luego esos datos se consolidan en un dashboard que responde preguntas de negocio como:
- ¿Cuál es el producto más vendido?
- ¿Qué producto es más rentable?
- ¿Cómo influye la satisfacción en las ventas?
- ¿El margen cubre los costos?
