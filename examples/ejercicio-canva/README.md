# Ejercicio 1. Mi tarjeta de presentación profesional

## Objetivo

Crear una tarjeta personal que pueda utilizarse para presentarse profesionalmente.

## Competencias Canva

* Plantillas
* Cuadros de texto
* Tipografías
* Colores
* Iconos
* Descarga

## Enunciado

Diseña una tarjeta de presentación profesional para ti mismo.

Debe incluir:

* Nombre y apellidos
* Profesión o perfil profesional
* Correo electrónico
* Teléfono ficticio
* Ciudad
* Código QR opcional

Investiga:

* Cómo buscar plantillas
* Cómo cambiar colores
* Cómo modificar fuentes
* Cómo insertar iconos

## Entregable

PNG y PDF.
