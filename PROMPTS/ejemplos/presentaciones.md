## PROMPT GENERICO CHHATGPT / GAMMA 

Copia este prompt y personalizalo een las secciones marcadas con []

```markdown
# Prompt básico para generar presentaciones en Gamma.app con ChatGPT

Actúa como un consultor experto en comunicación y diseño de presentaciones.

Quiero que generes el contenido completo para una presentación profesional en Gamma.app.

## Objetivo de la presentación
[TEMA DE LA PRESENTACIÓN]

## Público objetivo
[QUIÉN VA A VER LA PRESENTACIÓN]

## Objetivo final
[INFORMAR / VENDER / FORMAR / CONVENCER / INSPIRAR]

## Estilo deseado
- Visual
- Moderno
- Profesional
- Claro y directo
- Con ejemplos prácticos

## Estructura requerida
Genera:

1. Título de la presentación
2. Subtítulo
3. Índice
4. Entre 8 y 12 diapositivas
5. Para cada diapositiva incluye:
   - Título
   - Idea principal
   - 3-5 puntos clave
   - Sugerencia visual o imagen
   - Texto breve, no párrafos largos
6. Conclusión final
7. Diapositiva de cierre con llamada a la acción

## Requisitos adicionales
- Usa lenguaje claro y profesional
- Prioriza conceptos visuales y resumidos
- Incluye ejemplos reales cuando sea posible
- Añade metáforas o comparaciones si ayudan a entender el tema
- Evita exceso de texto
- Propón gráficos, iconos o esquemas visuales

## Formato de salida
Entrega el resultado en formato markdown estructurado para poder copiarlo directamente en Gamma.app.
```

Puedes reutilizar este prompt cambiando únicamente:

* Tema
* Público
* Objetivo

Ejemplo:

* Tema: “Transformación Digital en PYMEs”
* Público: “Directivos sin perfil técnico”
* Objetivo: “Convencer para iniciar un plan digital”


## Prompts con sintaxis lista para copiar

### 1. Prompt base para NotebookLM
```text
Eres un asistente que crea presentaciones en formato de diapositivas. 
Genera una estructura clara con título, objetivo, agenda, contenido principal, ejemplos y conclusión. 
Adapta el lenguaje para una audiencia técnica no experta.
```

### 2. Prompt para Gamma
```text
Necesito una presentación profesional en Gamma.
Crea diapositivas con un diseño visual simple y moderno, cada una con título, texto breve, puntos clave y sugerencia de imagen o gráfico. El tema es "Transformación digital en educación".
```

### 3. Prompt para NotebookLM con storytelling
```text
Genera una presentación que cuente una historia: problema, impacto, solución propuesta y resultados esperados. Incluye una diapositiva de llamado a la acción al final.
```

### 4. Prompt para Gamma con datos y gráficos
```text
Crea una presentación que explique datos clave sobre el crecimiento del marketing digital.
Incluye ideas para 3 gráficos: evolución anual, comparación de canales y proyección a 5 años.
```

### 5. Prompt para NotebookLM para público ejecutivo
```text
Prepara una presentación ejecutiva de 5 diapositivas sobre "Ventajas de implementar IA en pequeñas empresas". Usa lenguaje conciso, métricas relevantes y recomendaciones claras.
```

### 6. Prompt para Gamma para un taller
```text
Diseña una presentación tipo taller de 6 diapositivas sobre "Cómo preparar prompts efectivos". Incluye ejercicios prácticos y preguntas para la audiencia.
```

### 7. Prompt para NotebookLM con estructura educativa
```text
Crea una presentación didáctica para explicar "Qué es ChatGPT". Incluye definición, ejemplos de uso, ventajas, riesgos y consejos para comenzar.
```

### 8. Prompt para Gamma centrado en diseño
```text
Genera una presentación con estilo visual atractivo, fondo moderno y tipografía legible. El tema es "Tendencias 2026 en inteligencia artificial". Sugiere colores y tipos de elementos visuales.
```

### 9. Prompt para NotebookLM versión resumida
```text
Resume el contenido de un curso en una presentación de 4 diapositivas: resumen, puntos clave, aplicaciones prácticas y próximos pasos. Debe servir como material de cierre.
```

### 10. Prompt para Gamma colaborativo
```text
Crea una presentación colaborativa que facilite el trabajo en equipo. Incluye secciones para objetivos, roles, cronograma y métricas de éxito del proyecto.
```

### 11. Prompt para NotebookLM pregunta-respuesta
```text
Genera una presentación basada en preguntas frecuentes sobre "Uso responsable de IA". Cada diapositiva debe responder una pregunta clave con ejemplos claros.
```

### 12. Prompt para Gamma pitch
```text
Prepara un pitch deck breve para presentar una idea de negocio. Incluye problema, solución, mercado, modelo de negocio y propuesta de valor.
```
