# Biblioteca de Prompts para Educación

## 📚 Prompts para Docentes

### 1. Planificación de una lección
**Prompt:**
```
Eres un docente de [asignatura] para estudiantes de [grado/nivel]. Crea un plan de lección de [duración] minutos que incluya:
- Objetivos de aprendizaje claros
- Actividades principales (introducción, desarrollo, cierre)
- Materiales necesarios
- Evaluación formativa y/o cierre reflexivo
```

### 2. Generar actividades y ejercicios
**Prompt:**
```
Propón [número] actividades prácticas para reforzar el contenido sobre [tema]. Para cada actividad, ofrece:
- Instrucciones claras
- Tiempo estimado
- Criterios de éxito o rúbrica breve
```

### 3. Crear evaluaciones y cuestionarios
**Prompt:**
```
Diseña un cuestionario de [número] preguntas sobre [tema]. Incluye preguntas de tipo:
- Opción múltiple (con 4 opciones y respuesta correcta marcada)
- Verdadero/Falso
- Respuesta corta (1-2 frases)
```

### 4. Feedback formativo para estudiantes
**Prompt:**
```
Lee la siguiente respuesta de un estudiante y ofrece retroalimentación constructiva en tono amable, indicando qué está bien y cómo puede mejorar:

[respuesta del estudiante]
```

## 👩‍🎓 Prompts para Estudiantes

### 5. Resumir conceptos clave
**Prompt:**
```
Estoy estudiando [tema]. Haz un resumen claro y breve (máx. 5 frases) con los puntos clave que debo comprender.
```

### 6. Explicar con ejemplos
**Prompt:**
```
Explica el concepto de [concepto] con un ejemplo práctico y con un paralelismo sencillo que lo haga accesible.
```

### 7. Organizar ideas para un trabajo
**Prompt:**
```
Ayúdame a estructurar un trabajo sobre [tema]. Sugiere:
- Título y subtítulos
- Introducción breve
- Puntos principales a desarrollar (2-4)
- Conclusión sugerida
```

### 8. Motivación y técnicas de estudio
**Prompt:**
```
Dame 5 consejos prácticos para estudiar mejor en [tipo de materia] y mantener la motivación.
```

## 🧩 Prompts para Aprendizaje Personalizado

### 9. Seleccionar recursos educativos
**Prompt:**
```
Recomienda recursos (videos, artículos, libros cortos, actividades online) para aprender [tema] desde cero, en español.
```

### 10. Generar ejemplos y contraejemplos
**Prompt:**
```
Da 3 ejemplos correctos y 3 contraejemplos (errores comunes) sobre [tema] para ayudar a distinguir bien los conceptos.
```

## 💡 Cómo usar esta biblioteca
- Copia el prompt que mejor se adapte a tu necesidad.
- Sustituye los campos entre corchetes `[ ]` por el contenido específico.
- Si quieres que el prompt genere contenido en un estilo determinado (formal, lúdico, breve), añade esa instrucción al final.

¡Listo! Puedes seguir ampliando esta biblioteca con secciones como "Inclusión y diversidad", "Aprendizaje rápido" o "Gamificación" según tus necesidades.
