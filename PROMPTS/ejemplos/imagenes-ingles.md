# Anatomía de un Prompt

Un prompt completo suele estar formado por:

```text
[Sujeto]
+
[Acción]
+
[Entorno]
+
[Composición]
+
[Iluminación]
+
[Estilo]
+
[Calidad]
+
[Detalles adicionales]
```

Ejemplo:

```text
Astronauta
explorando
una ciudad abandonada
vista panorámica
luz de atardecer
estilo cinematográfico
ultra detailed
niebla y partículas flotando
```

# Plantilla Universal para practicar

```text
[SUJETO]
[ACTION]
[ENVIRONMENT]
[SHOT TYPE]
[LIGHTING]
[STYLE]
[MOOD]
[QUALITY]
```

Ejemplo rellenado:

```text
Young explorer,
walking through an ancient forest,
wide shot,
golden hour lighting,
fantasy illustration,
mysterious atmosphere,
ultra detailed,
high quality.
```

# Ejercicio muy potente para clase

Entrega esta tabla a los alumnos:

| Categoría | Opciones                          |
| --------- | --------------------------------- |
| Sujeto    | Astronauta, profesor, robot       |
| Acción    | caminando, explorando, observando |
| Entorno   | Marte, bosque, ciudad             |
| Plano     | primer plano, panorámico, aéreo   |
| Luz       | amanecer, atardecer, neón         |
| Estilo    | foto, anime, óleo                 |
| Emoción   | épico, alegre, misterioso         |

Luego les haces generar:

1. Una combinación aleatoria.
2. Una combinación diseñada por ellos.
3. La misma escena en tres estilos distintos.

Con este ejercicio entienden rápidamente que generar imágenes no consiste en "escribir una frase", sino en **controlar múltiples dimensiones visuales mediante vocabulario específico**. Esa es la base de la ingeniería de prompts para imagen.
