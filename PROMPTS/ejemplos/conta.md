# Prompts de Contabilidad - CONTA

Guía temática para elaborar prompts efectivos relacionados con contabilidad, finanzas y gestión contable.

---

## 1. Análisis de Estados Financieros

### Prompt Base
```
Analiza el siguiente estado financiero [INSERTAR DOCUMENTO] y proporciona:
- Interpretación de ratios clave (rentabilidad, liquidez, solvencia)
- Tendencias principales
- Áreas de preocupación identificadas
- Recomendaciones de mejora
```

### Variaciones Útiles
- **Por sector**: Especifica el sector (retail, manufactura, servicios)
- **Comparativo**: Incluye estados de múltiples períodos
- **Stakeholder**: Adapta el análisis para accionistas, bancos o gerencia
- **Problemas específicos**: "¿Qué significa este ratio negativo?"

---

## 2. Clasificación y Registro de Transacciones

### Prompt Base
```
Clasifica la siguiente transacción contable:
[DESCRIPCIÓN DE LA TRANSACCIÓN]

Proporciona:
- Cuentas afectadas (debe/haber)
- Clasificación contable
- Asiento contable necesario
- Justificación bajo [NORMA: NIIF/GAAP/LOCAL]
```

### Variaciones Útiles
- **Plan de cuentas específico**: Adjunta el catálogo de cuentas
- **Contexto tributario**: Incluye normativa local aplicable
- **Complejidad operativa**: Transacciones con múltiples fases
- **Revisión retroactiva**: Corrección de asientos anteriores

---

## 3. Elaboración de Presupuestos

### Prompt Base
```
Crea un presupuesto [PERÍODO: trimestral/anual] para [EMPRESA/ÁREA] considerando:
- Histórico: [DATOS HISTÓRICOS]
- Variables de crecimiento/decrecimiento: [%]
- Factores externos: [CONTEXTO ECONÓMICO]
- Restricciones: [LIMITACIONES]

Formato requerido: [ESPECIFICAR ESTRUCTURA]
```

### Variaciones Útiles
- **Escenarios múltiples**: Optimista, pesimista, probable
- **Presupuesto flexible**: Para diferentes niveles de actividad
- **Desglose por áreas**: Departamental, por proyecto
- **Sensibilidad**: Impacto de cambios en variables clave

---

## 4. Auditoría y Cumplimiento

### Prompt Base
```
Diseña un plan de auditoría para [ÁREA/PROCESO CONTABLE] que incluya:
- Riesgos principales identificados
- Procedimientos de auditoría recomendados
- Pruebas de cumplimiento
- Indicadores de anomalías a detectar
- Frecuencia de revisión
```

### Variaciones Útiles
- **Norma específica**: NIIF, PCGA, normativa local
- **Fraude**: "¿Dónde están los puntos vulnerables?"
- **Controles internos**: Evaluación y mejora
- **Post-auditoría**: Seguimiento de hallazgos

## 5. Impuestos y Obligaciones Fiscales

### Prompt Base
```
Analiza las implicaciones tributarias de:
[OPERACIÓN/TRANSACCIÓN]

Considerando:
- Jurisdicción: [PAÍS/REGIÓN]
- Régimen fiscal: [ESPECIFICAR]
- Normativa vigente: [AÑO]
- Estructuras alternativas: [SI APLICA]

Proporciona cálculos y recomendaciones.
```

### Variaciones Útiles
- **Deducibilidad**: ¿Qué es deducible y por qué?
- **Diferimientos**: Impuestos diferidos y provisiones
- **Declaraciones**: Ayuda en completar formularios
- **Optimización**: Dentro del marco legal
- **Cambios normativos**: "¿Cómo impacta la nueva reforma?"

## 6. Conciliaciones y Validaciones

### Prompt Base
```
Realiza la conciliación entre:
- [FUENTE A]: [DATOS/SALDOS]
- [FUENTE B]: [DATOS/SALDOS]

Identifica:
- Diferencias encontradas
- Posibles causas
- Asientos de ajuste necesarios
- Acciones preventivas
```

### Variaciones Útiles
- **Bancos**: Conciliación extracto-registros
- **Inventarios**: Físico vs contable
- **Cuentas por cobrar**: Antigüedad, incobrables
- **Mayor vs balanza**: Validación de saldos
- **Intercompañía**: Eliminación de transacciones

## 7. Informes Financieros Especializados

### Prompt Base
```
Genera un informe [TIPO: analítico/ejecutivo/técnico] sobre:
[TEMA CONTABLE]

Para audiencia: [STAKEHOLDER ESPECÍFICO]

Incluye:
- Resumen ejecutivo
- Análisis detallado
- Visualizaciones recomendadas
- Conclusiones y recomendaciones
- Apéndices técnicos (si aplica)
```

### Variaciones Útiles
- **Flujo de caja**: Proyecciones y análisis
- **Margen de contribución**: Por producto/línea
- **Punto de equilibrio**: Análisis CVU
- **EVA (Economic Value Added)**: Creación de valor
- **Dashboard financiero**: KPIs y métricas clave

## Consejos para Prompts Efectivos en Contabilidad

✓ **Sé específico**: Incluye cifras, fechas, normas aplicables
✓ **Proporciona contexto**: Tamaño empresa, sector, situación
✓ **Especifica formato**: Excel, PDF, diagrama, tabla
✓ **Indica normativa**: NIIF, PCGA, regulaciones locales
✓ **Define audiencia**: Usuario final del análisis
✓ **Sé claro en restricciones**: Presupuesto, tiempos, limitaciones
✓ **Adjunta datos relevantes**: Documentos, extractos, históricos
✓ **Indica urgencia**: Prioridad y fechas de entrega
